## Project Setup Instructions
To get started working on this project follow these steps:
* Create a directory and clone the reposetory in that directory.
* Next `cd` into the `server` directory and run `npm install` followed by `npm run nodemon`.
* Then `cd` into the `client` directory and run `npm install` followed by `npm run start`.
* Now the website should be ruuning on [localhost:8080](http://localhost:8080).
* And the api should be ruuning on [localhost:3000](http://localhost:8080).
